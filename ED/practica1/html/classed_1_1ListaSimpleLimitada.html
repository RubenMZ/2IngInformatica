template<class G>
class ed::ListaSimpleLimitada< G >

Clase para definir una lista limitada de elementos genéricos G.

Define una colección de elementos G organizados de manera secuencial. Cada elemento puede tener un anterior y un siguiente

estaVacia() implica not estaLlena()

estaVacia() implica numeroElementos() == 0
